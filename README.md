# EcoTrack Analytics

Dashboard profesional para monitorear huella ambiental, consumo energetico, proveedores y acciones sugeridas de sostenibilidad.

## Descripcion

EcoTrack Analytics simula una herramienta ejecutiva para empresas que necesitan medir y explicar su avance ambiental. El proyecto esta pensado para portafolio porque combina visualizacion de datos, filtros, tabla operativa, exportacion CSV y una interfaz responsive.

## Funcionalidades

- KPIs dinamicos de CO2e, energia renovable, ahorro estimado y riesgo proveedor.
- Grafico en canvas para emisiones mensuales.
- Filtros por sede y periodo.
- Tabla de proveedores con buscador y etiquetas de estado.
- Recomendaciones calculadas a partir de los datos.
- Exportacion de reporte en CSV.
- Diseno responsive para escritorio y movil.

## Tecnologias

- HTML5 semantico
- CSS3 responsive
- JavaScript vanilla
- Canvas API

## Como ejecutar

Abre `index.html` directamente en el navegador. No requiere instalacion ni servidor local.

## Valor para portafolio

Este proyecto demuestra criterio de producto, manejo de datos en frontend, visualizacion, experiencia de usuario para dashboards y una presentacion lista para explicar en entrevistas.
